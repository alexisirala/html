Se detallan con el vocabulario más cotidiano posible y las etiquetas que no se pueden explicar son enlazados a w3schools para profundizar.

  1. Comentarios: los comentarios se escriben en el siguiente formato en el siguiente Enlace , no son leídas por el userAgent o navegador.

  2. Doctype: el Doctype le informa al navegador que version de HTML usamos.

  3. Etiqueta a: La etiqueta a define un hipervínculo, que se utiliza para vincular de una página a otra. Y tiene una serie de Atributos como Download ,href y demás en el siguiente enlace tienes mas informacion para consultar como usarlo: Enlace

  4. Etiqueta abbr: esta etiqueta sirve para escribir acronimos y mostrar su significado atravez de un titulo, si necesitas mas informacion puedes consultar el siguiente Enlace . Un ejemplo practico sería UNA que el pasar el puntero encima te muestra el significado.

  5. Etiqueta address: La etiqueta address define un formato para ingresar una dirección un ejemplo en el area de pruebas:

  6. Etiqueta area: La etiqueta area define una etiqueta de imagenes donde se puede hacer selecciones con el mouse ,como usarlo: Enlace

  7. Etiqueta article: esta etiqueta se utiliza para dividir publicaciones y son usados normalmente en blogs y a veces en comentarios de los blogs ya que permiten dividir muy bien los comentarios. Enlace

  8. Etiqueta aside: se utiliza normalmente para las barras laterales de los blogs para mostrar contenido de la web o publicidad. Tienes mas info en el siguiente enlace

  9. Etiqueta audio: sirve para agregar audio a una pagina o sección. Un ejemplo en el enlace y en la zona de Ejercicios.

  10. Etiqueta base: Especifica una URL predeterminada y un destino predeterminado para todos los enlaces en una página, es decir en el head ponemos una url general como por ejemplo http://www.alexisirala.com.py/ y en el body solamente lo que sigue despues del ultimo slash de la dirección permitiendonos trabajar como si estuvieramos trabajando con archivos locales en el servidor.:enlace

  11. Etiqueta b: la etiqueta b pone en negrita el texto como por ejemplo hola soy un texto en negrita. enlace

  12. Etiqueta bdi: esta etiqueta lo que hace es aislar un texto que no queramos que se mueva con el css u otros formatos aplicados al html es decir una palabra estatica por ejemplo en un parrafo, puedes mirar un ejemplo enlace

  13. Etiqueta bdo: la etiqueta bdo permite cambiar la direccion de un texto ejemplo Texto . enlace

  14. Etiqueta blockquote: La etiqueta se utiliza para mostrar una cita de un autor por ejemplo y lo mostras de una manera diferente un ejemplo sencillo se encuentra sandbox y y enlace

  15. Etiqueta body: La etiqueta body es el cuerpo de la pagina, en el incluye todo lo que aparece dentro de la pagina. enlace

  16. Etiqueta br: es una etiqueta que nos da un salto de linea y se puede utilizar en cualquier parte del body o footer, ya sea para saltar una linea en un parrafo o imagen ,video o cualquier elemento del codigo html. enlace

  17. Etiqueta button: sirve para crear un boton que se puede clickear con el mouse un ejemplo pruebas y para mas informacion en el enlace

  18. Etiqueta canvas: esta etiqueta genera graficos que se puede manejar con css y javascript. enlace

  19. Etiqueta caption: sirve para trabajar con tablas. ejemplos en el sandbox y en el enlace
